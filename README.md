# user-activity-simulator

Микросервис для генерации и хранения данных о перемещении пользователей. Проект используется как вспомогательный модуль в экосистеме `ZOV` и позволяет имитировать пользовательскую активность, сохранять траектории движения и отдавать их по HTTP API.

## Что делает проект

- запускает симуляцию перемещения для набора пользователей;
- генерирует координатные смещения во времени;
- сохраняет историю движения пользователей в `unitdb`;
- позволяет остановить активные симуляции;
- возвращает накопленные данные по конкретному пользователю;
- поддерживает выборку части траектории через параметры диапазона.

## Технологии

- `Go`
- `net/http`
- `unitdb`
- `JSON API`
- `goroutines`
- `context`

## Запуск

```bash
go run main.go handler.go database.go simulator.go types.go
```

Доступные параметры запуска:

- `-dbpath` — путь к каталогу базы данных;
- `-addr` — адрес и порт HTTP-сервера.

По умолчанию сервис запускается на порту `8080`.

## API

### `POST /start`

Запускает симуляцию движения для списка пользователей.

Пример тела запроса:

```json
{
  "user_ids": ["user1", "user2", "user3"]
}
```

### `POST /stop`

Останавливает все активные симуляции.

### `GET /user/{user_id}`

Возвращает сохраненную историю движения пользователя.

Поддерживаются query-параметры:

- `min`
- `max`

Они позволяют получить только часть траектории в процентном диапазоне.
